## 💡 How to use the application

The interface is simple. After you launch the program, you will see a text box. Type your question directly into this box. 

For example, you can ask, "What are the main risks identified in Apple's latest 10-K?" or "How much did Microsoft spend on research and development last year?" 

The application retrieves relevant sections from the official documents and shows you the answer. It combines these findings with web search results to give you a complete picture.

## 🛠️ Features of the tool

This software helps simplify complex financial data. Here are the core features:

- **Document Analysis:** It reads SEC 10-K filings for major tech companies. This ensures accuracy based on official corporate disclosures.

- **Data Privacy:** Your search history remains on your local machine.
- **Efficient Retrieval:** The tool uses advanced search patterns internally to find the exact pages that contain your answer.
